# UI Framework

모바일 게임 UI 프레임워크.

## 예정 기능
- UIManager — 팝업 스택 관리 + Time.timeScale 제어
- UIPopup — 생명주기 훅 + UITransition 연동
- UITransition — Animator/DOTween 애니메이션
- SafeAreaFitter — 모바일 노치 대응
- NumberCounter — TMP 숫자 카운트 애니메이션
- ButtonClickEffect — 클릭 펀치 효과
- UIShake — 흔들림 효과
